# Student Attendance Tracker - Frontend

## Project Overview
This is the frontend for a student attendance tracking system built with React and TailwindCSS. It provides a UI for CRUD operations on attendance records.

## Features
- Display list of attendance records.
- Add new attendance.
- Edit existing attendance.
- Delete attendance.

## Tech Stack
- React (with Vite)
- TailwindCSS (CDN)
